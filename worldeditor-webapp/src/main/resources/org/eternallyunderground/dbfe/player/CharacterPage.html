[Create Armour Set From Tweak]

You can create a new armour set from your current Tweaked Appearance by entering a set name here and clicking Create. A tweaked appearance is created using /creatureTweak in Game when you have no items including bags equipped.

This set is refashioning only as it will have no stats. It will be added to your inventory.

The individual items names will be automatically created as you type the set name, but you may adjust these individually before creating the set.
